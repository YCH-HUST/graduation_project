# 4.4.3 数据库表设计

## 表1：用户表（auth_user）

| 字段名 | 描述 | 字段类型 | 是否为空 | 主键/外键 |
|--------|------|----------|----------|-----------|
| id | 用户ID | int | 否 | 主键 |
| username | 用户名 | varchar(150) | 否 | 唯一索引 |
| password | 密码（哈希） | varchar(128) | 否 | |
| role | 角色 | varchar(10) | 否 | |
| full_name | 真实姓名 | varchar(100) | 是 | |
| email | 邮箱 | varchar(254) | 是 | |
| gender | 性别 | varchar(10) | 是 | |
| age | 年龄 | int unsigned | 是 | |
| hospital | 所属医院 | varchar(100) | 是 | |
| department | 科室 | varchar(20) | 是 | |
| job_title | 职称 | varchar(50) | 是 | |
| years_of_experience | 从业年限 | int unsigned | 是 | |
| is_active | 是否激活 | tinyint(1) | 否 | |
| date_joined | 注册时间 | datetime | 否 | |
